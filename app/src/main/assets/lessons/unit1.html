ชนิดข้อมูลในภาษา JavaScript

      Data Types ในภาษา JavaScript ข้อมูลที่เราใช้ในการเขียนโปรแกรมจะมีประเภท (Type) กำหนดไว้ เพื่อให้คอมพิวเตอร์รู้ว่าจะจัดการกับข้อมูลเหล่านั้นอย่างไร ลองนึกภาพว่าชนิดข้อมูลเหมือนกับป้ายที่บอกว่าของในกล่องนี้คืออะไร จะได้เปิดและใช้งานได้ถูกวิธี JavaScript เป็นภาษาที่มีชนิดข้อมูลแบบ Dynamic Typing หมายความว่าคุณไม่ต้องประกาศ ชนิดข้อมูลของตัวแปรล่วงหน้า ตอนที่คุณใส่ค่าอะไรลงไปในตัวแปร JavaScript จะพิจารณาชนิดข้อมูลของค่านั้นเอง

1. Integer

Hello
World

2. String (ข้อความ)

ใช้เก็บข้อความ และสามารถใช้ ' ' หรือ " " หรือ ` `

let name = "John";
let greeting = `Hello, ${name}!`;

3. Boolean (ค่าความจริง)

ใช้เก็บค่า true หรือ false

let isAvailable = true;
let hasPermission = false;

4. Undefined

ค่าของตัวแปรที่ยังไม่ได้ถูกกำหนดค่า

let myVar;
console.log(myVar); // undefined

5. Null

ใช้แทนค่าที่ไม่มี หรือถูกตั้งใจให้ว่าง

let data = null;

6. Object (วัตถุ)

ใช้เก็บข้อมูลแบบ Key-Value

let person = { name: "Alice", age: 30 };

7. Array (อาร์เรย์)

ใช้เก็บข้อมูลหลายค่าในตัวแปรเดียว

let colors = ["red", "green", "blue"];

8. Function (ฟังก์ชัน)

ใช้สร้างโค้ดที่สามารถนำกลับมาใช้ใหม่ได้

function sayHello() {
return "Hello!";
}
